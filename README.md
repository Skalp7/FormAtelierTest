# FORM ATELIER

Premium digital atelier website built with Next.js App Router, TypeScript, Tailwind CSS, Framer Motion and Contentlayer MDX.

## Development

```bash
npm install
npm run dev
```

## Content

Project case studies live in `content/projects`. Add a new `.mdx` file with the existing frontmatter shape and it will appear in the project index and navigation automatically.

## Deployment

Cloudflare Pages build command:

```bash
npm run pages:build
```

Build output directory:

```bash
.vercel/output/static
```
